# PulseBrief - Draft Preview Guide

## Overview
This guide explains how to run a local preview build of PulseBrief for testing purposes. **No deployment is performed** as part of this preview process.

## Prerequisites
- Node.js and pnpm installed
- DFX (Internet Computer SDK) installed
- All dependencies installed (`pnpm install` in the frontend directory)
